Проєкт: CodeVerify

Платформа для перевірки коду з автоматичним тестуванням алгоритмічних задач

Ідея проєкту

Створення веб-платформи, яка дозволяє студентам та розробникам завантажувати рішення алгоритмічних задач і миттєво отримувати результати тестування. Система автоматично запускає код у безпечному середовищі та порівнює вихідні дані з очікуваними.

Функціонал

  • Система автентифікації: Особисті кабінети для студентів та викладачів.
  • Каталог задач: Розподіл завдань за рівнями складності (Easy, Medium, Hard).
  • Автоматичне тестування: Підтримка перевірки результатів у реальному часі.
  • Статистика: Відстеження прогресу виконання та часу, витраченого на задачу.

Структура сайту

  • Головна сторінка: Загальна інформація та останні новини.
  • Dashboard: Панель керування з переліком доступних завдань.
  • Сторінка задачі: Опис умови, вікно для завантаження файлу або вводу коду.
  • Профіль: Історія успішних здач та рейтинг користувача.
